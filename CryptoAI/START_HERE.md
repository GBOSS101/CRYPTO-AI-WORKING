# Quick Start Scripts

## Start Web Dashboard (Recommended)
```powershell
python dashboard.py
```
Then open your browser to: **http://127.0.0.1:8050**

## Start CLI Version
```powershell
python main.py
```

## Features Comparison

### Web Dashboard (dashboard.py)
- ✅ Modern, interactive web interface
- ✅ Real-time updates every 30 seconds
- ✅ Beautiful charts and visualizations
- ✅ Multiple tabs for different features
- ✅ Easy trade execution with forms
- ✅ Live market sentiment indicators
- ✅ Responsive design

### CLI Version (main.py)
- ✅ Terminal-based interface
- ✅ Menu-driven navigation
- ✅ Colored output
- ✅ Lightweight and fast
- ✅ Good for remote/SSH access
